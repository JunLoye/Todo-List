![截图](image/README/1782566211462.png)

## 快速开始

**下载预构建版本**（推荐）：从 [Releases](https://github.com/junloye/Todo-List/releases) 下载 `TodoList.exe`，双击即可运行。

**从源码构建**：

```bash
npm install
npm start          # 启动开发模式
npm run dist       # 打包为安装包
```

## 功能

- 任务增删改查，支持截止日期与优先级
- 习惯追踪与 OKR 目标管理
- 多主题（浅色/深色/多彩）与紧凑模式
- 数据导入/导出，任务归档
- 自动更新检查

## 技术栈

Electron 28.x · 原生 HTML/CSS/JS · localStorage · Electron IPC
